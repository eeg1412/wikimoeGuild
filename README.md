# ⚔️ 冒险公会 — 放置类公会经营策略游戏

> 建立你的公会，招募冒险家，征服无尽迷宫，称霸竞技场！

## 🎮 游戏简介

**冒险公会**是一款融合了放置养成、策略对战与自由交易的 Web 端多人在线游戏。你将扮演一名公会长，从零开始经营属于自己的冒险者公会——招募冒险家、探索深渊迷宫、收集强力符文石、在竞技场中与其他玩家一决高下。

无论你是挂机党还是策略控，这里都有适合你的玩法。

---

## ✨ 核心特色

### 🏰 地下迷宫 · 放置产出

- 将冒险家派遣到迷宫，**自动产出四种水晶**，无需时刻盯守
- 切换迷宫刷新水晶产出概率，寻找最佳收益组合
- 挑战迷宫军团，击败 Boss 升级迷宫，解锁更高等级的符文石

### ⚔️ 冒险家 · 深度养成

- 招募多达 **50 名**冒险家，每位拥有独特元素属性与战斗偏好
- 四维属性（攻击/防御/速度/SAN）自由升级，打造专属战斗风格
- **元素克制系统**（水→火→风→地→水，光明↔黑暗）为战局带来策略博弈

### 💎 符文石 · 收集与合成

- 三种稀有度：**普通 / 稀有 / 传说**，技能与增益随机生成
- 打造专属 Build：主动技能 × 被动增益，每颗符文石都独一无二
- 分解、升级、合成——循环利用，让每一颗符文石物尽其用

### 🏆 竞技场 · PVP 对战

- **赛季制竞技**，在 5×5 棋盘上排兵布阵
- 匹配相近实力的对手，积累竞技点冲击排行榜
- 赛季奖池丰厚，前三名瓜分巨额金币奖励

### 🏪 交易市场 · 玩家经济

- **官方市场**：稳定的水晶收购与出售渠道
- **自由市场**：与其他玩家自由交易水晶、碎片与符文石
- 低买高卖，靠商业头脑也能富甲一方

### ⛏️ 矿场 · 扫雷探险

- 切换迷宫时随机发现矿场，**全服玩家共享**
- 10×10 扫雷玩法，挖掘水晶、挑战隐藏军团赢取稀有奖励
- 成为矿主，其他玩家挖矿时你也能**坐享收益**

## 🛠️ 技术栈

| 层级     | 技术                                       |
| -------- | ------------------------------------------ |
| 前端     | Vue 3 + Vite + Tailwind CSS + Element Plus |
| 后端     | Node.js + Express                          |
| 数据库   | MongoDB + Mongoose                         |
| 实时通信 | SSE（Server-Sent Events）                  |
| 部署     | Docker + Docker Compose                    |

## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/your-repo/wikimoeGuild.git
cd wikimoeGuild

# 安装依赖
pnpm install

# 启动开发服务器
cd server && node app.js      # 后端
cd client && pnpm run dev     # 前端
```

或者使用 Docker 一键部署：

```bash
docker-compose up -d
```
